Basic Bash Commands

ls: List Files (Show files)

pwd: Print working directory (where am I?)

clear: clears the screen

cd: change directory (to go inside a directory)

cd .. : back button (to come out of a directory)

absolute path: starts with '/' and will work on every location no matter kis directory k andar h hum (eg: '/desktop/shaurya')

relative path: do not starts with '/' and will not work everywhere (eg: 'desktop/shaurya')

'/': root directory

'~': home directory

cd ../.. : will take you two steps back
