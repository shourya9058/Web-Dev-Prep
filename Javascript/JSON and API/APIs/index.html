AJAX (Asynchronous JavaScript and XML):

AJAX ek technique hai jo web pages ko bina refresh kiye data fetch karne ya bhejne ki suvidha deti hai. Yeh asynchronous hota hai, matlab ek request bhejne ke baad page ka baaki ka content block nahi hota.

Http Verbs:

  • GET: Jab humein sirf server se data fetch karna ho, tab GET request use karte hain.
  • POST: Jab humein server ko koi naya data bhejna ho (e.g., form submission, user registration), tab POST request use karte hain.
  • DELETE: Jab humein server se koi specific data delete karna ho, tab DELETE request use karte hain.

Status Codes:

Hrr request k baad data k saath ek code bhi return hota which have a meaning to it.

Example:

  • 200: Everything is OK
  • 404: Not found (API/page doesn't exist)
  • 400: BAd Request
  • 500: Internal Server Error
